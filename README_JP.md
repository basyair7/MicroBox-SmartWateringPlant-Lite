# MicroBOX スマート自動散水システム Lite
Blynk アプリ対応 IoT 自動植物散水システム（Lite版）

## バージョン
- プロジェクトバージョン：1.0

## 対応ボードタイプ
### ボードタイプ １
- ボード：nodemcuv2 (NodeMCU)
- マイコン種別：ESP8266
- フレームワーク：Arduino
- ハードウェアバージョン：1.0.0
- ソフトウェアバージョン：1.0.1

### ボードタイプ ２
- ボード：esp32doit-devkit-v1 (ESP32)
- マイコン種別：ESP32
- フレームワーク：Arduino
- ハードウェアバージョン：1.0.1
- ソフトウェアバーション：1.0.2
